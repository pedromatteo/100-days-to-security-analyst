# Day 09 - Linux File System & Basic Commands

## Objectives
- Learn the Linux filesystem structure.
- Practice basic file and directory manipulation.
- Understand absolute and relative paths.
- Become comfortable navigating through the terminal.

## Topics Studied
- pwd
- ls
- cd
- mkdir
- touch
- cp
- mv
- rm
- cat
- less
- whoami
- id
- Absolute vs Relative Paths

## Practical Activities

Created files and directories:

- notes.txt
- logs.txt
- backup/

Practiced:

- Copying files using `cp`
- Moving and renaming files using `mv`
- Removing files using `rm`
- Listing directories using `ls`
- Navigating directories with `cd`
- Viewing current directory using `pwd`
- Creating files with `touch`
- Investigated a real HTTP connection using Wireshark.
- Identified source and destination IP addresses.
- Identified source and destination MAC addresses.
- Analyzed TCP flags (SYN, ACK, PSH, FIN).
- Interpreted an HTTP GET request and a 200 OK response.

## Reflection

Today reinforced the Linux commands that are used daily by security analysts. File manipulation and navigation became much more natural after practicing directly in the terminal.
